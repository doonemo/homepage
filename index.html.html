Defense Specification

국방 가이드라인 만족을 위한
독자적 솔루션 기능 설계

두네모 솔루션은 국방망 특화 환경 및 까다로운 육·해·공군 보안 규정을 즉각 바인딩할 수 있는 **차세대 국방 전용 아키텍처(AI Harness Framework)**를 제품 자체 핵심 기능으로 완전 구현하여 탑재하고 있습니다.

요구사항 입력을 시작으로 소스코드 자율 생성, SAST/DAST 자동 점검 중계, 3단계 게이트 승인, 그리고 마지막 형상관리(.hwpx 군 표준 설계서 자동화 추출 및 GitLab 형상관리망 안전 연동)에 이르는 전 주기의 라이프사이클 관리를 타사 모듈 추가 없이 통합 솔루션 기능만으로 안정적으로 처리합니다.

nemoAI 플랫폼 특화 4대 보유 기능

1
MCP 기반 이기종 모듈 플러그인 통합 기능

상용 SAST, Sandbox/DAST, 사설 지식 데이터 RAG 등 이기종 솔루션을 원클릭으로 결합·제어하는 MCP 통신 엔진 내장

2
국방망 전용 독립 배포 프로파일 설계 기술

동일 상용 코어 바이너리를 유지하면서 '나'급 국방 보안 설정 및 무결성 검증 필터를 물리 분리해 동적 주입하는 기능 지원

3
3단계 Human-in-the-Loop 결재 승인 콘솔 탑재

개발자(수락) ➡ 보안담당자(취약점 검토) ➡ 최종책임자(배포 승인)로 연계되는 국방 최적화 4대 뷰포트 결재 시스템 기본 탑재

4
군 표준 .hwpx 및 GitLab 형상관리 자동화 모듈

검증 완료된 소스코드와 보안 검사 내역을 통합하여 군 표준 한글 포맷 문서(.hwpx)로 자율 변환하고 형상관리망으로 이송하는 연동 모듈 기본 탑재
